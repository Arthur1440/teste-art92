# Teste Art92 - GitHub Page

Uma GitHub Page criada com HTML, CSS e JavaScript estático.

## 📋 Sobre

Este é um projeto de exemplo para demonstrar como criar uma GitHub Page com conteúdo estático. A página é totalmente responsiva e funciona em todos os dispositivos.

## 🚀 Como funciona

1. **HTML** - Estrutura semântica da página
2. **CSS** - Styling moderno com gradientes e animações
3. **JavaScript** - Interatividade com scroll suave e validação de formulário

## 🌐 Acessar a página

Sua GitHub Page está disponível em:
```
https://arthur1440.github.io/teste-art92
```

## 📁 Estrutura dos arquivos

```
teste-art92/
├── index.html      # Página principal
├── styles.css      # Estilos CSS
├── script.js       # Scripts JavaScript
└── README.md       # Este arquivo
```

## ✨ Recursos

- ✅ Design responsivo (mobile-first)
- ✅ Navegação com scroll suave
- ✅ Seções: Hero, Sobre, Projetos, Contato
- ✅ Animações e transições
- ✅ Formulário de contato
- ✅ Footer

## 🛠️ Como editar

1. Abra o arquivo `index.html` para editar o conteúdo
2. Modifique `styles.css` para alterar cores e layouts
3. Adicione funcionalidades em `script.js`
4. Faça commit e push - a página atualizará automaticamente!

## 🎨 Personalizando

### Cores
Altere as cores no arquivo `styles.css`:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
}
```

### Conteúdo
Edite o HTML em `index.html` com seu conteúdo próprio.
